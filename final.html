Examination Instructions

Welcome — VMC Coding Assessment

This is a timed coding assessment designed to evaluate your programming skills and problem-solving abilities. Please read all instructions carefully before proceeding.

Note: This is a proctored examination. Your activity will be monitored throughout the test duration. Proceed only from the designated examination centre.

Test Guidelines

  • You will have 60 minutes to complete all problems
  • Problems may be solved in any order
  • Each problem has visible and hidden test cases
  • You may run your code multiple times before final submission
  • Ensure you submit before time expires — auto-submission is not guaranteed

Proctoring & Security Rules

✖Do not switch tabs or minimize the browser
✖Do not use any external websites or resources
✖Do not communicate with any person during the test
✖Do not use AI tools, code generators, or online compilers
✖Do not copy-paste code from any external source
Warning: Violation of any rule will result in immediate disqualification and forfeiture of candidature.

Technical Requirements

  • Ensure a stable internet connection throughout the test
  • Use Chrome, Firefox, or Edge (latest versions recommended)
  • Close all unnecessary applications and browser tabs
  • Disable extensions that may interfere with the platform
  • Test in a quiet environment free from distractions

By clicking "I Agree, Start Test" you confirm that you have read, understood, and agree to comply with all examination rules and guidelines.
