Welcome to VMC Coding Assessment

This is a timed coding assessment designed to evaluate your programming skills and problem-solving abilities. Please read the following instructions carefully before starting the test.

⚠️ Important: This is a proctored examination. Your activity will be monitored throughout the test duration.

📋 Test Guidelines

  • You will have 60 minutes to complete all problems
  • You can solve problems in any order
  • Each problem has public and private test cases
  • You can run your code multiple times before final submission
  • Make sure to submit your solution before time runs out

🔒 Proctoring & Security Rules

  • Do not switch tabs or minimize the browser window
  • Do not use any external resources or websites
  • Do not communicate with anyone during the test
  • Do not use any AI assistance tools or code generators
  • Do not copy-paste code from external sources
⚠️ Warning: Any violation of these rules will result in immediate disqualification and your test will be terminated.

✅ Technical Requirements

  • Ensure you have a stable internet connection
  • Use a modern browser (Chrome, Firefox, or Edge recommended)
  • Close all unnecessary applications
  • Disable browser extensions that may interfere
  • Test in a quiet environment without distractions

By clicking "I Understand, Start Test", you acknowledge that you have read and agree to follow all test guidelines and rules.
